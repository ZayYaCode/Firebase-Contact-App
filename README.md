A sleek and responsive contact management application built with **React**, **Tailwind CSS**, and **Firebase**. This app allows users to manage their contacts with real-time database updates and a clean user interface.

## ‚ú® Features

- **Add Contacts**: Create new contact entries with Name and Email.
- **Read Contacts**: View a list of all saved contacts in real-time from Firebase.
- **Update Contacts**: Edit existing contact information through a modal interface.
- **Delete Contacts**: Remove contacts with an instant success notification.
- **Search Functionality**: Quickly find specific contacts using the integrated search bar.
- **Success Notifications**: Instant feedback using Toast notifications for every action.
- **Responsive Design**: Fully functional on both desktop and mobile devices.
